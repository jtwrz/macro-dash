Starter Project — Architecture

Yield Curve & Inflation Dashboard · how the pieces connect
① Data source
FRED API — Federal Reserve Economic Data
Free API, instant signup. Returns JSON time series for any of 800k+ economic series. You call it with a series ID and a date range; it returns an array of date/value pairs.
DGS3MO → DGS30 (Treasury yields) DFII5 → DFII30 (TIPS real yields) T5YIE · T10YIE (breakeven) CPIAUCSL · PCEPI · PPIACO · CPILFESL (inflation) CES0500000003 (wage growth) free · rate-limited to 120 req/min
↓
② Fetch layer — src/fred.js
fetchSeries(id, startDate, units)
Calls one FRED series. Strips null observations. Returns a clean [{date, value}] array. Handles the units=pc1 flag for YoY % change on inflation series.
vanilla JSES moduleasync/await
fetchMultiple(ids, startDate, units)
Loops over an array of series IDs, calls fetchSeries for each with a small delay between calls, returns a keyed object — one call to load all yield curve series at once.
vanilla JSES modulerate-friendly delay
③ Chart modules — src/
yieldCurve.js
Fetches Treasury, TIPS, and breakeven series. Groups into monthly snapshots (first business day). Renders a Chart.js line chart plotting yield vs maturity. Drives the animation: play/pause, scrubable slider, 0.5×/1×/2×/4× speed, series toggles. Stat cards (10Y yield, spreads, breakeven) update as you scrub.
Chart.jsanimation loopmonthly snapshots
inflationPanel.js
Fetches 6 inflation series with units=pc1. Renders a multi-line time series chart. Each series gets a distinct color + dash pattern. Horizontal dashed line at 2% (Fed target). Custom HTML legend above chart shows series names and current values pinned to right edge.
Chart.jsmulti-lineYoY % transform
↓
④ Page shell — index.html
index.html — the entire app in one file
Loads Chart.js from CDN (no install needed). Two-panel layout: yield curve left, inflation right. Stacks to single column on mobile below 900px. Shows a loading skeleton while FRED data fetches. Imports both chart modules and initialises them on DOMContentLoaded. Header shows title + last-updated timestamp.
Chart.js via cdnjs CDN no framework no bundler no build step dark #0B1220 background responsive

⑤ Deploy — .github/workflows/deploy.yml
GitHub Actions
Workflow triggers automatically on every push to main. No build step — it just deploys the repo root directly. Runs on GitHub's servers, not your machine. Free within GitHub's generous free tier.

Open the URL on your phone, laptop, or work computer. On page load the browser fetches fresh data from FRED, renders both charts, and you're looking at live economic data. No login, no app to install, no server to babysit.
